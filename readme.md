# これはなに
立ち絵素材を重ねて表示できるツールです。  
どんな表情にしようかな～って考えるときに使えるかもしれません。  
ちょっと便利なsurfaces.txt作成補助つき。

# ダウンロード
ダウンロードはこちらからできます。  
リンク先、下の方にある「SurfaceBuilder.zip」をクリックするとダウンロードできます。
